Use Tonictypes Paginator Html

You can use tonictypes paginator html as easy-to-use implementation for the pagination information, that is added to the template variables. Use the following code snippet in your template to include a paginator. 


  <dv:template.render template="EXT:tonictypes/Resources/Private/Templates/Pagination/Paginator.html" arguments="{
    paginator:p_paginator,
    paging:p_paging,
    pages:p_pages,
    itemsPerPageVariableName:p_items_per_page_var_name,
    pageNumberVariableName:p_page_number_var_name,
    addQueryString:1
  }"/>
